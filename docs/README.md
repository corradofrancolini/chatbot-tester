## Quick Links

### Installation
```bash
./install.sh
```

### New project
```bash
./chatbot-tester --new-project
```

### Run tests
```bash
./chatbot-tester --project=<name> --mode=train
```

### Help
```bash
./chatbot-tester --help
```
